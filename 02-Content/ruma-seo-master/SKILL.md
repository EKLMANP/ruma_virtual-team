## 3. 核心工作流程

### Phase 1: 關鍵字研究

1. **使用 `search_web` 搜尋目標主題**：分別搜繁中 & 英文關鍵字
2. **分類關鍵字**：

| 類型 | 數量 | 使用位置 | 範例 |

| 主關鍵字 | 1 個 | 標題 H1、首段、結尾 | 龍舟新手 |
| 次要關鍵字 | 3-5 個 | H2 標題、內文 | 龍舟體驗、划龍舟、龍舟教學 |
| 長尾關鍵字 | 5-10 個 | H3 標題、FAQ | 第一次划龍舟要準備什麼 |

3. **評估關鍵字競爭度**：優先挑選搜尋量高但競爭度中低的詞組

## 4. SEO 最佳化規範

### 4.1 標題標籤 (Title Tag)

```
格式：[主關鍵字] - [吸引點擊的描述] | RUMA 龍舟隊
長度：最多 60 字元（繁中約 30 字）

✅ 好的範例：
"龍舟新手完整入門指南：從零開始的划槳教學 | RUMA 龍舟隊"

❌ 避免：
"RUMA龍舟隊部落格文章關於龍舟的各種資訊給新手參考"
```

### 4.2 Meta Description

```
格式：[說明內容] + [價值主張] + [CTA]
長度：150-160 字元

✅ 好的範例：
"第一次划龍舟不知道怎麼開始？RUMA 龍舟隊整理新手必知的裝備、姿勢、訓練技巧，
帶你從零建立正確觀念。立即了解並預約免費體驗！"
```

### 4.3 URL 結構

```
格式：https://rumadragonboat.com/news/[簡短關鍵字]

✅ 好的範例：
/news/dragon-boat-beginner-guide
/news/taipei-dragon-boat-experience

❌ 避免：
/news/post-12345
/news/2024/01/18/article
```

### 4.4 標題層級結構

```markdown
# H1：主標題（頁面唯一，包含主關鍵字）
## H2：主要段落標題（包含次要關鍵字）
### H3：子段落標題（包含長尾關鍵字）
#### H4：細節說明
```

### 4.5 內部連結策略

每篇文章至少包含 2-3 個內部連結：
- 連結至 RUMA 課程報名頁
- 連結至相關文章（已發布的）
- 連結至關於我們頁面

## 5. AEO (Answer Engine Optimization)

### 5.1 Position Zero 優化

在文章開頭 2-3 句話直接回答使用者問題：

```markdown
**使用者搜尋：** "龍舟新手需要準備什麼？"

**開場撰寫方式：**
> 龍舟新手需要準備的基本裝備包括：快乾運動服、水上運動鞋、
> 防曬用品和替換衣物。划槳和救生衣通常由龍舟隊提供，
> 初學者無需購買專業器材。
```

### 5.2 FAQ Schema 結構

每篇文章必須包含 5-8 個 FAQ：

**Schema Markup 建議：**
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "第一次划龍舟會很難嗎？",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "龍舟對新手非常友善！基本動作在 30 分鐘內就能學會..."
    }
  }]
}
```

### 5.3 HowTo Schema (適用教學文)

```json
{
  "@context": "https://schema.org",
  "@type": "HowTo",
  "name": "如何正確划龍舟",
  "description": "龍舟划槳基本技巧教學",
  "step": [{
    "@type": "HowToStep",
    "name": "入水階段 (Catch)",
    "text": "將槳垂直插入水中..."
  }]
}
```

### 5.4 定義框

對專業術語提供明確定義：

```markdown
> **什麼是龍舟？**
> 龍舟是一種源自中國的傳統長型船隻，通常可容納 10-22 名划手，
> 前端設有鼓手、後端設有舵手，是端午節的傳統競技項目。
```

### 5.5 表格整理

將比較性資訊整理成表格：

```markdown
| 龍舟類型 | 人數 | 長度 | 用途 |
|---------|------|------|------|
| 標準龍舟 | 22人 | 12.4m | 正式比賽 |
| 小龍舟 | 10人 | 9.14m | 練習/小型比賽 |
```
